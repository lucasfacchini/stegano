# Descrição #

Uma implementação da técnica de esteganografia para esconder mensagens em imagens, escrito em Python.

Esta é uma entrega de trabalho para a disciplina de Processamento de Imagens ([Universidade de Caxias do Sul](https://www.ucs.br/)), desenvolvido por [Lucas Facchini](https://github.com/lucasfacchini) e [Cassiano Carraro](https://github.com/CassianoCarraro).
